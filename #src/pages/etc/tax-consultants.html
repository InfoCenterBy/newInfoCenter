Наши услуги

Для индивидуальных предпринимателей

Для получения услуги при себе необходимо иметь паспорт и свидетельство о регистрации индивидуального предпринимателя.
  • Консультирование по вопросам налогообложения
  • Составление налоговых деклараций (расчетов) для ИП
  • Составление отчетов в ФСЗН, Белгосстрах работодателями
  • Подготовка рекомендаций (заключений) по вопросам налогообложения
  • Оформление пакета документов, связанных с исчислением заработной платы за месяц
  • Месячное пакетное обслуживание для ИП

Для физических лиц

Для получения услуги при себе необходимо иметь паспорт.
  • Консультирование по вопросам соответствия доходов и расходов граждан
  • Составление пояснений по вопросам соответствия доходов и расходов граждан
  • Методологическая помощь физическим лицам в составлении пояснений об источниках доходов
